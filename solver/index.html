To add a constraint, type one of the following in the box:

1) A guess result. Use lowercase for incorrect (gray) letters, uppercase for correct (green) letters, and for misplaced (yellow) letters, precede them with a period. If you wish to omit a letter (meaning that anything can go into that spot), use a dash.

Example: .ap-L-

2) A letter count. Type a letter, a comparison operator (<, =, >), and a number.

Example: e > 0

3) Show only common words. Literally type "_common" (without the quotes)

You must have at least one guess result for results to show up. This may be simply a string of dashes.
